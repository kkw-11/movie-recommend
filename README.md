# 🎬 Movie Recommendation System

> **TF-IDF 기반 콘텐츠 필터링 추천 시스템**  
> 사용자가 선택한 영화를 분석하여 유사한 영화를 추천하는 웹 서비스
---

## 🎯 프로젝트 소개

### 문제 정의

영화 추천 서비스는 대부분 **사용자 평점 데이터**에 의존합니다. 하지만 신규 사용자나 평점 데이터가 부족한 경우 **Cold Start 문제**가 발생합니다.

### 솔루션

**Content-Based Filtering**을 통해 사용자 히스토리 없이도 영화의 **메타데이터(장르, 감독, 배우, 줄거리)**만으로 추천을 제공합니다.

### 핵심 가치

- ✅ **Cold Start 문제 해결**: 신규 사용자도 즉시 사용 가능
- ✅ **실시간 처리**: 평균 응답 시간 0.2초
- ✅ **해석 가능성**: 추천 이유를 설명 가능 (Feature 기반)
- ✅ **확장성**: FastAPI 전환으로 비동기 처리 가능

---

## ✨ 주요 기능

### 1. 영화 검색
- TMDB API 연동을 통한 실시간 영화 검색
- 한글/영어 검색 지원
- 포스터, 평점, 줄거리 등 상세 정보 제공

### 2. 맞춤형 추천
- 사용자가 선택한 3개 영화 분석
- TF-IDF 기반 유사도 계산
- 20개의 유사 영화 추천

### 3. 반응형 UI
- 직관적인 카드 기반 인터페이스
- 모바일/태블릿/데스크톱 지원
- 실시간 검색 (Debounce 적용)

---

## 🛠 기술 스택

### Backend
- **Flask 3.0.0**: 웹 프레임워크 (FastAPI 마이그레이션 준비 중)
- **Python 3.9+**: 핵심 언어
- **scikit-learn**: TF-IDF, Cosine Similarity
- **NumPy**: 수치 연산

### Data Source
- **TMDB API**: 영화 메타데이터 (20만+ 영화)
- **데이터 수집**: 200개 인기 영화 (확장 가능)

### Infrastructure (로컬 개발)
- **환경 관리**: python-dotenv, virtualenv
- **API**: RESTful 설계
- **배포 준비**: Docker, GCP (예정)
